# Installation

Redstone api is published in **npm**, so you can install it using npm or **yarn** cli tool.

{% tabs %}
{% tab title="npm" %}
```bash
npm install redstone-api
```
{% endtab %}

{% tab title="yarn" %}
```bash
yarn add redstone-api
```
{% endtab %}
{% endtabs %}
